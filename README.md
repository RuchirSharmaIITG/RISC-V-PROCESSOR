# RISC-V-PROCESSOR

## Overview
This repository contains a custom 5-stage pipelined RISC-V processor implemented in RTL. The core architecture fully supports the base integer instruction set (RV32I) and has been extended to include the single-precision floating-point extension (RV32F) for advanced arithmetic operations. 

## Key Features
* **5-Stage Pipeline:** Efficient instruction execution with hazard handling.
* **RV32I + RV32F Support:** Integer base set with advanced arithmetic.
* **Custom UART Logic:** Integrated for reliable serial communication and seamless debugging.
